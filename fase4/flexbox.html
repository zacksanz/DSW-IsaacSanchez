Flex box

Este es un modelo de diseño concebido para ayudar a distribuir el contenido a lo largo de un eje.
Básicamente, consta de un contenedor flex que en cuyo interior se anidan los elementos que se van disponiendo en una dirección y sentido específicos.

Para configurar un contenedor como contenedor flex, se debe establecer la propiedad CSS del elemento correspondiente display: flex-box

Algunas otras propiedades CSS que se deben tener en cuenta a la hora de configurar correctamente un contenedor flex son:

  • flex-direction: Establece la dirección del flujo de los elementos hijos. Los valores que puede adquirir son row (por defecto) para distribuir los elementos de forma horizontal y column para distribuirlos verticalmente. También existen las opciones row-reverse para distribuir el contenido de derecha a izquierda y column-reverse para disponerlo desde abajo hacia arriba.
  • flex-wrap: En algunos casos, debido a la extensión del contenido de un flex box, es deseable disponer de más de una fila o una columna para mostrar el contenido. Un contenedor flex box puede adaptarse para mostrar este contenido en más de una fila o columna, dependiendo de la dirección que se haya establecido. Para este caso, se debe establecer la propiedad flex-wrap: wrap (que por defecto viene configurada como flex-wrap: nowrap). Como alternativa, existe la opcion flex-wrap: wrap-reverse.

Para establecer los elementos del contenedor flex, basta con que este esté dentro de un contenedor flex. Algunas propiedades a tener en cuenta a la hora de personalizar la apariencia de un elemento de un contenedor flex son

  • order: Establece el orden de aparición de los elementos de un contenedor. Los valores válidos que puede adquirir son números enteros.
  • flex-grow, flex-shrink y flex-basis: Permiten configurar la forma como se redimensionan los elementos dentro del contenedor flex.
