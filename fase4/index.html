Propósito del sitio

Orientar al lector sobre las características y prestaciones que ofrecen los conceptos de diseño de Flex Box y CSS Grid para la construcción de sitios web.

Introducción

Existen dos modelos de diseños de páginas web que particularmente trataremos en este sitio. El primero de ellos, se trata de un contenedor tipo flex. Este es un modelo de diseño concebido para ayudar a distribuir el contenido a lo largo de un eje.
Básicamente, consta de un contenedor flex que en cuyo interior se anidan los elementos que se van disponiendo en una dirección y sentido específicos. El segundo modelo es más familiar a los diseñadores gráficos, ya que se trata de configurar una retícula, una especie de rejilla invisible en la que se podrán disponer los contenidos, posicionándolos en filas o columnas.
