CSS Grid

En este modelo de diseño obtenemos una retícula o rejilla bidimensional (a diferencia del flex box) en la que podemos distribuir los elementos contenidos que hacen parte del diseño.

Las propiedades más importantes a tener en cuenta para configurar correctamente un contenedor grid son:

  • grid-template-rows: Establece la distancia vertical entre cada una de las líneas que delimitan las filas. Los valores descritos se separan por un espacio en blanco. Ejemplo:
    grid-template-columns: auto 12px 13mm 15cm 3em
  • grid-template-columns: Establece la distancia horizontal entre cada una de las líneas que delimitan las columnas. Ejemplo:
    grid-template-columns: auto 12px 13mm 15cm 3em
  • grid-gap, grid-row-gap y grid-column-gap Establece el espacio entre las filas y/o columnas.
